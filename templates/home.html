What We Offer

Everything you need to understand your survey data

A complete toolkit that maps raw survey column codes to human-readable field names, block references, question text, and coded value labels — in one unified table.

CSV Microdata Ingestion

Upload any survey microdata CSV. The app auto-detects encoding, extracts unique level values, and reports column structure instantly.

Excel Layout Mapping

Parses the Data Layout Excel to map every column code (e.g. b11c4) to its full field name, block number, item reference, and byte position.

PDF Schedule Codebook

Extracts "CODES FOR BLOCK N" sections from the Schedule PDF, matching coded values (1 = Male, 2 = Female…) to every relevant column.

Searchable Metadata Table

A filterable, sortable table lets you quickly find any variable, inspect its question text, and view all value labels in one place.

Analytics & Reports

Generate interactive Plotly charts, statistical summaries, and AI-powered insights. Export full HTML or PDF reports.

One-Click Export

Download the consolidated metadata repository as a clean CSV — ready for documentation, codebook generation, or downstream analysis.
